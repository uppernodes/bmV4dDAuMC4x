# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability
If you found a vulnerabilty, please don't exploit it.
Send it to our security team at security@uppernodes.com ASAP!
